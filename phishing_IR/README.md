# Phishing Incident Response Case Study

## Objective
This project demonstrates how a SOC analyst investigates and responds to a phishing email incident.

## Incident Overview
A suspicious email was reported by an employee claiming an urgent password reset request.

## Skills Demonstrated
- Incident Response
- Email Analysis
- Threat Intelligence
- SOC Documentation

## Tools Used
- Email Header Analysis
- Open Source Threat Intelligence (OSINT)

## Outcome
The phishing attempt was identified, contained, and preventive measures were recommended.

## Working Component

This project includes a Python-based phishing email analyzer that:
- Reads a suspicious email file
- Extracts URLs and email addresses
- Detects phishing keywords
- Provides a phishing risk verdict

This simulates how a SOC analyst performs initial phishing triage.
